Welcome to 5eCS, a DM Combat Program. This is a combat program for AD&D v5.5.

Copyright 2005+, Portions copyright Wizards of the Coast (see the Open Gaming Licence)

It is not a character generator or a campaign program ... just combat. 

Hopefully it will be quick & easy to use. I'm able to setup a large dungeon in 30mins.

This program will not do everything, most are too complicated for that but I've tried to go for the 80/20 Rule ... 
80% of the rules for 20% of the effort. So, I'm lazy ...

Windows will not remove all the files when you uninstall. Files will exist in "C:\Program Files (x86)\5ecs-win" and
"C:\Users\{username}\AppData\\Local\VirtualStore\Program Files (x86)\5ecs-win" which you may need to delete manually.

Please be aware that if you use animated icons they can slow the system down alot. And good animated icons are hard to find. The best of luck to you.

Only png/jpeg/gif & animated gif picture types are supported. I have included several of the free token & map collections that I could find on the internet.
I have converted them to jpeg (smaller by 75%) and moved them around a bit.

You can have many campaigns or groups running using this program. First you need to Copy & Paste the Desktop Icon using Right-Click.
Then Rename it using a Right-Click. Then Right-Click once more, select Properties, and add the campaign name (no spaces please) to the
end of the Target field (it will have 5eCS.exe in it). Click OK and then when you double click it should use that folder to save/load any details.

Each "map" can have upto 256 monsters and 70 encounter groups defined. You can have many maps and monster files that you can load/save from the System tab.

You can use this as just a DM combat aid, but if you attach an extra screen, you can use that to show the players screen/map either at the 
table or virtually using zoom/signal etc.

Ok, how to use this wonderful program ....

Go to the System tab
  Set Monster & Map file names to whatever is appropriate
  Set the System Flags/Options

  Set the map size in the System tab as required and press the New button if you've changed the size.

  Make 5eCS bigger using the two slider fields "Screen Width" & "Screen Height". You can not drag it bigger or maximize it.
  This is a problem with the GUI library used.

- Clear all the existing information by going to the Qk Heal/XP/Enable tab.
  In the Special Actions area, select "Remove ALL Players" and press "Do Action". 
  Then select "Reset for a new adventure" and press "Do Action".

- Add all the players in the Edit/Add tab.
  I also add their spell attack details as a weapon with the DC details in the "Special Message" field. It makes things easier IMHO.

  Players can also have an alternate form. For instance, a druid could wildshape into a bear. So, first add that as a player
  but set it to Disabled in the "Flag/Options" area. Take note of the Interal ID (just to the right of the name). Now, edit that
  player, in the "Alt Form ID" field (just below AC) set that ID value from before. Take note of this players Internal ID.
  In the Alternate Form player, set it's "Alt Form ID" to be the ID value of the actual player. Basically you're setting a linked list
  so you could even have several alternate forms for a player ie. player -> AF1 -> AF2 -> AF3 -> player.
  You can switch forms in the Combat tab using the pop-up menu by selecting "Switch to Alt Form".

- Go to the Combat tab to do the map. You can either draw it using the colors/textures or load a battle map picture (png/jpeg/gif).

  To load a battle map, Right click on where you want the map to start and select "Object, notes or Map Images" from the menu.
  You can either load a pre-defined map or use the File button to load it from your computer.
  Select "Map Image" to mark it as a map image.
  If you have a copy of the map that has all things DM removed like secret doors etc set that in the Player Map field.
  You can make the map bigger or smaller by changing the "Grid Pixel Size" value.

  To block "line of sight" on the battle map, change the "Map Mode" (middle LHS) to Freestyle, select the second last color (blue) and then 
  draw around each of the walls. You should see those grid squares getting bordered in a blue color.
Change the "Map Mode" back to Combat when you're done. To remove these, draw over the "walls" with the
  first color (white) using the steps as above.

  If you want to change the color of that "color", right click on it and select the one you want.

  If you want to draw the map, set the "Map Mode" (middle LHS) to one of the drawing modes, select a color and away you go.
  Take care which color slot you select as some are 'solid' and others not ... check the tip that appears.
  I find that if I'm doing tunnels or the like, using a Shape/Circle 5' works very nicely.
  If you draw with the wrong 'color' then you can swap them for another by first taking note of the from/to color number 
  which is at the end of the 'tip' the go to the System tab and down the bottom you can swap the colors over.
  There is an undo function in the "Sp. Action" menu on the top right of the map.
  Set the "Map Mode" (middle LHS) to Combat once you have finished.

  You can also add objects/notes to the map, Right click on where you want the object and select "Object, notes or Map Images" 
  from the menu. You can use the Search tab to find icons that contain that text. Change its size in the "Icon Size" fields.
  You can also have hidden objects ie. trap icons that you unhide once they are triggered.

  You can add many maps to the combat area. I just draw "walls' around them to block the players from seeing that map.

- For each combat/encounter area:
    Make sure the "Map Mode" on the Combat tab is set to Combat

    Add the monsters for that area in the Edit/Add tab. There are lots standard monsters already loaded ready for you to
    use. Use the "Qk Monster Search" area to find and select them. Dont forget to assign an icon to them before adding them.

    If they are a non-standard monster, use either the "Qk Monster Add" or select "New Monster" and fill in all the details.
    You can also have an Inactive icon for a monster, so if you have something like a mimic, you can set the "Disabled Icon"
    field to be a chest/door and then when you 'activate' it, it becomes a monster.

    In the Combat tab select "Add NEW monsters to an Enc. Group" in the popup menu (by right clicking on the
    location to add these monsters). Type in the room/group name and select "Add them as Disabled" as then they
    will not appear in any of the main lists throughout the program.

    I also put details into the Note field like trap details, PDF page no's etc

    You can add up to 70 encounter groups/rooms. If you're doing more than that then select "Move all NEW monsters here" once you've
    reach the limit.

    Move each monster to their combat location.

To move around the map, you can:
  - zoom in/out using the buttons or the mouse wheel
  - right click and centre the maps from the menu options
  - double click on a location and it will centre the map there
  - click & drag your way around the map image

To do a combat:
  Move the players & monsters to their locations. Either drag the icons or use the popup menu.
  If you've added them in as a group, Right click on one and select "Enable just this Enc. Group" & "Unhide this Enc. Group"
  This will disable & unhide all the monsters except this group which will be enabled. Otherwise just Enable & Unhide each monster.

  You can also Hide one or more monsters in that group if you only want some of them to "appear" on the player screen.

  In the Initiatives tab select "New Combat" and add the players initiatives rolls to their numbers. Then "Accept Initiatives"
  to setup the Init Lists.

  Go to the Combat tab.

  If you have a second monitor attached, press the "Show XM" button and drag the player window to the other monitor. Maximize it.
  on the DM screen, zoom in/out both windows as required and select "Centre BOTH Maps here" from the popup menu.

  You'll see the initiative sequence on the left. As each one does their action they'll change color. You can select the one
  doing the current action by right-clicking on them either on the map or on the initiative sequence list.

  You can also show/hide the Init Seq. on the player screen by selecting that option in the pop-up menu. It will show the players
  and monsters that are enabled & not hidden/dead.

  On the player screen, Player and monster HP status percentages are shown in blocks of 20% so 75% will be shown as 60%.

  Once you've completed all the actions for that round, press the "End of Round" button. You can either use the "Do an Action"
  selection to do a complex action or "Do a Quick Attack" to just attack. I generally use the Quick Attack to do most things.

  If you want to add extra monsters into the combat, just add & move them in as above, then in the Initiatives tab, select "Show new Monsters/Refresh"
  and you should see them appear in the "Monster Initiatives" area. Then select "Accept Initiatives" to have them added to the combat.

  If spells are cast that modify "things" like Bless/Aid etc, add these into the "Modifiers" tab. The "Quick Modifiers" area is the easiest 
  place to do this but if you have more complex ones then add them into using the "Complex/System Modifiers".

  You can also add Alerts for the DM in the "Alerts/Names/Treasure/NPC" tab.

  To do mass player healings; in the "Qk Heal/XP/Enable" tab there's the Quick Heal area. Just refresh the list, type in the healing
  against each player and accept it.

Other procedural steps:
- after each 'day', you should select the "Reset all for a new Day" in the System tab. This will clear all the alerts, adjustments,
  heal them by level HP's etc. It pretty much clears/resets everything ready for a new day of adventuring
- in the "Qk Heal/XP/Enable" tab in the middle there is a list of special actions you can do

To do a Mass Combat
- when you add the army units, make sure you set the "#In Group" field before you add them. This number will then appear on the bottom RHS
  of the icon.
- just right-click anywhere on the combat grid and select "Do Mass Combat". Select who's attacking whom, "Calculate" and then adjust
  the defenders details based on how many have died. If you have hurt 'units' then I just set the current HP's to what's left and next
  time around remove it from that first.
	NB: the "# Died" adjusts the '#In Group' value by that amount but the 'HP:Curr' will set it to that.

Notes/quirks:
	- if you put the mouse pointer onto a player/monster on the map, in the "Combat/Map Messages" it will show you their details.
	- if you left click on two points on the map, in the "Combat/Map Messages" it will show you the distance between those points.

	- if you draw with the wrong 'color' then you can swap them for another by first taking note of the from/to color number 
	  which is at the end of the 'tip' the go to the System tab and down the bottom you can swap the colors over.

	- if you want to mess with your players heads, put random object icons (pentagrams etc) around your map and watch them scurry for cover :)

   - you can also draw false secret doors on the map which have Glyph of Wardings on them so they soon get trained to ignore them on the map

   - if you want to copy objects to say make a tree forest, select the Map Mode "Copy Object", select the one to copy and then where-ever
     you click, a copy will appear. Changes to one will effect all.

	***********************
	*** Menu/Tab Layout ***
	***********************

Combat - this is the main map area. Here you can draw a map, move players and monsters around and do the actual combat.
	Moving around is by either using the scroll bars, arrow keys or double-clicking on a spot.

Initiatives
	New Combat/Initiatives - do this when you're starting a brand new combat. Add the players rolls to the values shown.
	Show new Monsters - if you've added more monsters or enabled some, then selecting this will show them. You'll need to 'Accept' next.
	Accept Initiative - accept the initiative rolls as listed. If someone delays their action, then change their value and 'Accept' them.

	All changes will be reflected in the Initiative List on the combat tab.

Enc. Groups - do group actions like assigning members to a group. Enabling all in a group etc.

Edit/Add - the main area for adding or changing players/monsters. You can add something by:
		- selecting the appropriate "New .." in the lists if you're doing a one off monster
		- select "Qk Monster Add" adding a number of the same monster. It will only ask for the necessary details. Don't forget to 
		  assign an icon to it. Useful if you're adding them from a purchased module.
		- there are over 840 monsters from the SRD and the like preloaded in the "Qk Monster Search" area. Type in the first few characters
		  of its name, select how many you want and assign an icon to be used then "Add them". There are also prerolled NPC characters in here
		  like Cleric 8th level and the like.

Other
	Auto Damage - here you can do automatic damage like ones that are done from Melfs Acid Arrow and the like.
	Mass Skill Check - seaching a room or seeing if the players spotted your ambush. Do a secret roll using either a rerolled d20 (from the Edit
		  tab) or use a random d20.
	Log File - if you've enabled the logging option in the System tab, you'll see all the details here.
	Qk Icon - forgotten to assign icons to all those giants, then select and assign an icon here.
	Mass Copy - this allows you to copy all or some of the details from one monster to another.
	Qk Spells - generate random spell lists.
	Monster Heal - do mass healings on monsters.

System - This is where you set program options, file names for saving/loading players, maps & monsters etc
   If you want to convert to using d100 systems, then select them from the Dice System field and change the Initiative System as required.
   Note: if you do this you'll need to "Save All" and restart the program as it modifies all the screens

Notes - write your campaign notes in here. It will be saved when you do a "Save All" in the System tab.

Dice Roller - roll random dice silently.

Status - show the status of all active ie. not disabled players and monsters.

Spell Adjustments
   Advanced - this is where you add adjustments of any kind. Use a predefined one or add a new one. Such things as Prayer, Bless,
	   Bears Endurance and the like, basically anything that effects BAB, Saves, HP, Damage etc can be configured here. Select who it effects,
   	duration and the parts to the adjustment ie. +3 BAB Morale, +2 Damage Enhancement.
   Quick - just put the value of the adjustments in the appropriate column for that player. Useful if they have time to pre-buff before combat.

SRD Text - if you've extracted a HTML SRD to the 'srd' directory then it will show in there. These are good ones to use are:
	http://www.systemreferencedocuments.org/35/stephenh.zip
	http://www.darkshire.net/~jhkim/rpg/srd/srd_revised_html/v35SRD_html.zip

	If you extract the SRD, it will look thro the 'spell' directory and build up a list for you to use in the Combat section.
   So when you do an action you can quick search and review the spell details.

Qk Heal/XP/Enable
	Quick Heal - after a combat there's lots of healing, so it's easy to do it here. It shows their real HP's ie. no adjustments.
	XP Calc - calculate how much XP the players have gotten from defeating the active monsters ie. those enabled.
	Special Actions - do such actions like clear, reset player & monsters.
	Enable/Disable - mass enable or disable monsters.

Alerts/Name/Treasure/NPC
	Alerts - setup DM alerts that appear in the End of Round messages. Such things as when extra monsters appear or the like.
	Rnd Treasure - generate random treasure or specific items.
	Generate Rnd names - generate random names.
	NPC creation - generate a NPC and then add them as a monster.
